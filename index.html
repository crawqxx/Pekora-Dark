How to Install Using Stylus

Follow these steps to install the pekora.zip dark theme with the Stylus extension:

  • 1. Install the Stylus extension for your browser.
  • 2. Click the "Download Now" button to visit the site where the code is provided or click the "Download Direct" for direct download.
  • 3. If "Download Now" button was clicked, you can check out more information, and after that click "Install with Stylus"
  • 4. You should get redirected to Stylus extension, press "download" at the left and you're good to go!
